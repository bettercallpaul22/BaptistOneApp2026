# Forum Detail Page Redesign Plan

## Overview
Redesign the forum detail page to fetch real posts from the API, render them as cards, and support infinite scroll pagination.

## Files to Modify

### 1. `src/services/forum/forumService.ts` - Update ForumPost type
Replace the current `ForumPost` interface with the actual API response shape:

```typescript
export interface ForumPostAuthor {
  id: string;
  type: string;
  displayName: string;
  avatarUrl: string | null;
  firstName: string;
  lastName: string;
}

export interface ForumPostMediaFile {
  id: string;
  url: string;
  type: string;
}

export interface ForumPost {
  id: string;
  forumId: string;
  authorType: string;
  authorId: string;
  title: string;
  content: string;
  postType: string; // "ANNOUNCEMENT" | "GENERAL" | etc.
  moderationStatus: string;
  moderationReason: string | null;
  moderatedAt: string | null;
  moderatedByProfileId: string | null;
  mediaFileIds: string[];
  isPinned: boolean;
  isLocked: boolean;
  createdAt: string;
  updatedAt: string;
  author: ForumPostAuthor;
  mediaFiles: ForumPostMediaFile[];
}
```

Keep `ForumPostsResponseData` as is (it already matches `{ items, meta }`).

### 2. `src/store/slices/forumSlice.ts` - No changes needed
The `fetchForumPostsThunk` already exists and works correctly with the service layer.

### 3. `src/pages/forum/ForumDetailPage.tsx` - Redesign UI

**Header section:**
- Replace the large card with a simpler header matching Events page pattern:
  ```
  [MessageSquare icon]  Forum Title
                        Forum description
  ```
- Remove the "Forum context" section (department/unit info)
- Remove the "Forum summary" sidebar

**Post cards:**
Each post card should show:
- Author avatar (using `post.author.avatarUrl` or `post.author.displayName`)
- Author name (`post.author.displayName`)
- Timestamp (`post.createdAt` formatted)
- Post title (`post.title`) if present
- Post content (`post.content`)
- Post type badge (`post.postType` - e.g., "ANNOUNCEMENT")
- Pinned indicator if `post.isPinned`

**Layout:**
```
┌─────────────────────────────────┐
│ [Back button]                   │
│                                 │
│ [MessageSquare] Forum Title     │
│ Forum description text          │
│                                 │
│ [Posts] [Members]  ← tabs      │
│ ─────────────────────────────── │
│                                 │
│ ┌─────────────────────────────┐ │
│ │ [Avatar] Author Name    2h  │ │
│ │ Post Title                  │ │
│ │ Post content text here...   │ │
│ │ [ANNOUNCEMENT] badge        │ │
│ └─────────────────────────────┘ │
│                                 │
│ ┌─────────────────────────────┐ │
│ │ [Avatar] Author Name    5h  │ │
│ │ Another post title          │ │
│ │ More content...             │ │
│ │ [GENERAL] badge             │ │
│ └─────────────────────────────┘ │
│                                 │
│ [Load more / Scroll sentinel]   │
│                                 │
└─────────────────────────────────┘
```

**Key implementation details:**
- Use `post.author.displayName` for author name
- Use `post.author.avatarUrl` for avatar image
- Format `post.createdAt` with `new Date(post.createdAt).toLocaleDateString()`
- Show `post.postType` as a styled badge (rounded-full, bg-[#EEF4FF], text-[#123B8D])
- Show pin icon if `post.isPinned`
- Use IntersectionObserver with `loadMoreRef` for infinite scroll (already implemented)

### 4. Remove unused imports
- Remove `forumData.ts` related imports (`findForum`, `findDepartment`, `findUnit`)
- Remove `AppCard` if not used for posts
- Keep `AppStateFeedback` for loading/error/empty states
