# Plan: Implement Forum PostList Screen

## Summary
Create a new ForumPostListPage that shows posts for a clicked forum. Re-add navigation from ForumPage cards. All Redux infrastructure (thunk, slice state, service) already exists — just need the UI page, route, and navigation.

## Existing Infrastructure (no changes needed)
- `src/services/forum/forumService.ts` — `getForumPosts(forumId, page, limit, includePending)` already exists
- `src/store/slices/forumSlice.ts` — `fetchForumPostsThunk` + state (`posts`, `postsMeta`, `postsLoading`, `postsLoadingMore`, `postsError`, `postsLoadMoreError`, `currentForumId`) all exist
- `src/pages/forum/forumData.ts` — mock data (not needed for this)

## Changes

### 1. Create `src/pages/forum/ForumPostListPage.tsx`
New component following established patterns (ForumPage, WalletTransactionsPage):
- Use `<AppShell>` wrapper
- Back button navigating to `/forum`
- Fetch forum details from `state.forum.items` by `forumId` param
- On mount, dispatch `fetchForumPostsThunk({ forumId, page: 1, limit: 20 })`
- Read `posts`, `postsMeta`, `postsLoading`, `postsError`, `postsLoadMoreError` from `state.forum`
- Infinite scroll via `IntersectionObserver` on sentinel ref (same pattern as ForumPage)
- Render each post as a card: author avatar, author name, post type badge, title, content, created time
- Loading/error/empty states using `<AppStateFeedback>`
- Post type badge colors: ANNOUNCEMENT=gold, GENERAL=blue, PRAYER=green, EVENT=purple

### 2. Re-add paths to `src/routes/paths.ts`
Restore `forumDetails` and `forumDetailsRoute`:
```ts
forumDetails: (forumId: string) => `/forum/${encodeURIComponent(forumId)}`,
forumDetailsRoute: '/forum/:forumId',
```

### 3. Register route in `src/routes/index.tsx`
- Add lazy import: `const ForumPostListPage = lazy(() => import('@/pages/forum/ForumPostListPage'));`
- Add route inside `<ProtectedRoute>`: `<Route path={paths.forumDetailsRoute} element={<ForumPostListPage />} />`

### 4. Re-add navigation to `src/pages/forum/ForumPage.tsx`
Restore `useNavigate` import, `paths` import, `const navigate = useNavigate()`, and the `onClick`/`onKeyDown` handlers on the forum card `<div>`.
